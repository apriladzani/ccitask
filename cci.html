Apa Itu HTML, CSS dan JavaScript?

Mini Task CCI

🌐

HTML

HTML atau bisa dikatakan Hypertext Markup Language adalah fondasi dari sebuah website atau HTML bisa dikatakan struktur sebuah website.

Digunakan untuk apa sih HTML?

  • Untuk membuat struktur website, tanpa adanya html website tidak akan bisa dibuat atau dilihat oleh user

Apa yang dilakukan oleh HTML?

  • Untuk membuat struktur dasar pada website
  • Untuk membuat konten pada website
🎨

CSS

CSS atau bisa dikatakan Cascading Style Sheets adalah bahasa yang digunakan untuk memberikan kehidupan (UI) atau mengatur style, display, animasi pada website.

Digunakan untuk apa sih CSS?

  • Untuk membuat website menjadi lebih menarik dan responsif

Apa yang dilakukan oleh CSS?

  • Untuk mengatur tampilan website seperti warna, font dan ukuran
  • Untuk mengatur animasi website seperti hover
  • Untuk mengatur layout website seperti grid dan flexbox
⚡

JavaScript

JavaScript adalah bahasa pemrograman untuk membuat website menjadi interaktif atau singkatnya JavaScript memberi kehidupan (fungsional) pada website.

Digunakan untuk apa sih JavaScript

  • Digunakan untuk menambahkan interaktivitas pada website

Apa yang dilakukan oleh JavaScript

  • Untuk membuat website menjadi lebih interaktif seperti memberikan efek visual
  • Untuk merespon tindakan user seperti mengklik tombol, mengisi form, mengirim data
